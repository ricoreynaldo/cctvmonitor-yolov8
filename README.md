# CCTV Human Detection System Using YOLOv8

## Overview
This project is a real-time human detection application using the YOLOv8 algorithm and PyQt5 graphical user interface (GUI). The system is designed for CCTV monitoring and object detection visualization.

The application is capable of detecting human objects from CCTV or video input in real time and displaying the detection results through an interactive desktop interface.

---

## Features
- Real-time human detection
- YOLOv8 object detection model
- PyQt5 graphical user interface
- CCTV/video input support
- Screenshot capture feature
- Video recording feature
- Detection visualization with bounding boxes
- Multiple input source support

---

## Technologies Used
- Python
- YOLOv8
- OpenCV
- PyQt5
- Ultralytics
